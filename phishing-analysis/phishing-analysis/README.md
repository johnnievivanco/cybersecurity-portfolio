# Phishing Analysis

## Overview

Simulated a phishing investigation by analyzing suspicious emails to identify malicious links, spoofed domains, and social engineering techniques.

## Tools Used

* Simulated email samples
* Browser-based link analysis
* Online domain verification tools

## What I Did

* Analyzed suspicious emails for phishing indicators
* Examined sender addresses and domains for spoofing
* Inspected embedded links for deceptive or malicious URLs
* Identified social engineering techniques such as urgency and impersonation
* Documented findings and assessed potential risk

## Findings
* The sender email address (`support112@paypa1.net`) uses a spoofed domain that mimics a legitimate PayPal address
* The message uses urgency (“24 hours” and “account permanently disabled”) to pressure the user into taking immediate action
* The email uses generic language (“Dear PayPal customer”) instead of addressing the user directly
* The link encourages users to “confirm information,” which is a common phishing tactic used to steal credentials
* The presence of an unexpected attachment (`gG.pdf`) increases the risk of malware delivery


## Business Impact

Phishing attacks can lead to credential compromise, unauthorized access, and data breaches. Identifying these threats early helps protect sensitive information and reduces organizational risk.

## Why It Matters

Phishing is one of the most common initial access methods used by attackers. Detecting and analyzing phishing attempts is critical for preventing security incidents and maintaining user and system integrity. This type of phishing attack is commonly used as an initial access vector in real-world security incidents and can lead to account compromise and data breaches.

## Key Takeaways

* Learned how to analyze email content, links, and sender information for phishing indicators
* Developed the ability to identify common social engineering techniques
* Improved understanding of how phishing attacks are used to gain initial access in real-world environments
