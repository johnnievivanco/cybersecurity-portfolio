# Phishing Analysis

**Overview:**  
Analyzed simulated phishing emails to identify suspicious links, domains, and social engineering techniques.

**Tools Used:**  
- Email samples (simulated)  
- Browser analysis  
- Online domain verification tools

**What I Did:**  
1. Reviewed email headers and metadata for authenticity.  
2. Checked URLs for spoofing or malicious activity.  
3. Identified phishing red flags in subject lines and content.  
4. Documented risks and prevention strategies.

**Findings:**  
- Several emails used domain spoofing to appear legitimate.  
- Detected common patterns in phishing attempts, such as urgency or fear tactics.

**Key Takeaways:**  
- Developed attention to detail for identifying phishing threats.  
- Learned how to communicate findings clearly for end-users or leadership.  
- Practiced creating actionable recommendations to mitigate phishing risks.
