# Incident Response Playbook

**Overview:**  
Created a simulated incident response playbook for handling security events such as brute-force attacks or malware infections.

**Tools Used:**  
- Documentation templates  
- Security frameworks (NIST, SANS)  
- Sample incident data

**What I Did:**  
1. Defined incident response workflow: detection, containment, investigation, eradication, recovery.  
2. Created step-by-step playbook for SOC analysts to follow during an incident.  
3. Added escalation paths for severe incidents to leadership.  
4. Documented communication templates for internal and executive reporting.

**Key Takeaways:**  
- Learned the full lifecycle of incident response.  
- Practiced documenting technical processes for team use.  
- Developed skills for bridging technical work to leadership reporting.
