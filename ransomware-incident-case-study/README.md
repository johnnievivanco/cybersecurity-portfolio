# Ransomware Attack → Endpoint Compromise

> This project is part of my cybersecurity portfolio focused on incident response and threat detection.

---

## Overview

This case study simulates a ransomware infection affecting a user endpoint and demonstrates incident response procedures in a high-impact security scenario.

---

## Scenario

A user reported inability to access files, followed by detection of abnormal file encryption activity and suspicious processes on the system.

---

## Detection

- Rapid file encryption and abnormal file extensions  
- Suspicious processes running on the endpoint  
- User-reported system disruption  

---

## Triage

- Identified encryption patterns consistent with ransomware behavior  
- Observed system performance degradation  
- Validated activity as a high-severity security incident  

---

## Investigation

- Analyzed endpoint behavior to identify the initial infection vector  
- Checked for lateral movement or spread to other systems  
- Confirmed compromise limited to a single endpoint  

---

## Containment

- Isolated the affected endpoint from the network  
- Terminated malicious processes  
- Disabled network connectivity to prevent lateral spread  

---

## Recovery

- Restored affected data from backups  
- Reimaged system if necessary  
- Implemented additional monitoring to ensure full eradication  

---

## Outcome

- Threat contained to a single system  
- No evidence of spread across the network  
- System restored to secure operational state  

---

## Skills Demonstrated

- Incident response workflow  
- Endpoint threat analysis  
- Containment and recovery strategy  
- High-severity incident handling
